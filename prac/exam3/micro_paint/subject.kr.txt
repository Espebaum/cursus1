작업 파일을 읽고 터미널에 결과를 출력하는 프로그램을 작성하시오.

프로그램 작업 파일의 경로를 담는 인자 하나만을 받아야 합니다.
만약 인자가 없거나, 하나보다 더 많은 인자가 주어지면 당신의 프로그램은 개행을 포함한 "Error: argunment"를 표준 출력해야 합니다.

$> ./micro_paint | cat -e
Error: argument$
$> ./micro_paint do not fear mathematics | cat -e
Error: argument$
$> 

작업 파일을 열거나 읽는 도중 발생하는 모든 문제에 대해서는 개행을 포함한 "Error: Operation file corrupted"을 표준 출력하십시오.

작업 파일은 한 줄당 하나의 명령어를 포함하고 있습니다.
명령 줄이 정확하지 않다면 에러가 발생합니다.
프로그램에 에러가 발생하면 1을 반환하십시오.
에러가 없다면 0을 반환하십시오.
마지막 줄은 개행이 있거나, 혹은 없을 수 있습니다.
반드시 순서대로 명령 줄을 읽어야 하며, 따라서 실행 또한 동일한 순서로 이루어집니다.
한 명령 줄에 있는 변수 사이에는 적어도 하나의 공백이 있어야만 합니다.

WIDTH HEIGHT BACKGROUND_CHAR
이 명령 줄은 반드시 파일의 첫 번째 줄에 위치하며 그릴 위치(Draw Zone)를 정합니다. 당신의 프로그램은 draw zone 바깥에는 아무것도 표시해서는 안됩니다.
- WIDTH : 0보다 크고 300보다 작거나 같은 정수(int)여야 합니다. draw zone에 사용될 x축(horizontal number)입니다.
- HEIGHT : 0보다 크고 300보다 작거나 같은 정수(int)여야 합니다. draw zone에 사용될 y축(veritcal number)입니다.
- BACKGROUND_CHAR : 나머지 빈칸은 BACKGROUND_CHAR로 채워질 것입니다.

r X Y WIDTH HEIGHT CHAR
이 작업은 빈 직사각형을 그릴 것입니다. 직사각형의 경계선 부분만 그립니다.
- r : 캐릭터 r.
- X : float 형태, 직사각형 좌상단의 수평 위치입니다.
- Y : float 형태, 직사각형 좌상단의 수직 위치입니다.
- WIDTH : 0이 아닌 양수 float, 직사각형의 너비입니다(수평).
- HEIGHT : 0이 아닌 양수 float, 직사각형의 높이입니다(수직).
- CHAR : 직사각형을 그릴 때 쓸 캐릭터.

R X Y WIDTH HEIGHT CHAR
이 작업은 꽉 찬 직사각형을 그릴 것입니다.
- R : 캐릭터 R.
- X : float 형태, 직사각형 좌상단의 수평 위치입니다.
- Y : float 형태, 직사각형 좌상단의 수직 위치입니다.
- WIDTH : 0이 아닌 양수 float, 직사각형의 너비입니다(수평).
- HEIGHT : 0이 아닌 양수 float, 직사각형의 높이입니다(수직).
- CHAR : 직사각형을 그릴 때 쓸 캐릭터.

draw zone은 각각의 구역에 하나의 캐릭터를 담을 수 있는 직사각형들로 구분되어 있습니다(왜냐하면 우리가 터미널 안에 있으니까요).
우리는 그것을 픽셀이라 부를 것입니다. 쉽게 말해 우리는 어떤 픽셀이 직사각형인지 판단하기 위해 그 픽셀의 좌상단만을 사용할 것입니다.
직사각형의 경계선으로부터 1보다 크거나 같은 길이를 가진 좌상단의 픽셀은 빈 직사각형의 일부분이 아닙니다.
직사각형의 경계선으로부터 1보다 작은 길이를 가진 좌상단의 픽셀은 빈 직사각형의 일부분입니다.

당신은 다음 몇 줄을 읽으면서 이미지를 보아야만 합니다(micro_paint.png). 그것은 터미널을 나타냅니다. 우리는 WIDTH 5와 HEIGHT 3의 draw zone을 세팅했습니다.
우리는 당신에게 다음 설명을 쉽게 이해시켜주기 위해 다른 문자들을 사용했습니다. (파일 참조)
- 당신이 다음 작업을 하고 싶다면 : r 2.5 1.5 1 1 X (초록색 직사각형)
-- 이는 이미지 안의 문자 D가 X로 대체된다는 것을 의미합니다.

당신은 몇몇 opeariton_file.example로 당신의 코드를 테스트하기 위해 our_micro_paint를 찾아야 합니다.

힌트:
지점이 (Xa, Ya)로 정해져있고
좌상단이 (Xtl, Ytl), 우하단이 (Xbr, Ybr)인데
만약 Xtl <= Xa <= Xbr이고 Ytl <= Ya <= Ybr이면 지점이 직사각형 안에 있는 것입니다.
